# 🧮 macOS Calculator Clone

A glassmorphism-style calculator inspired by macOS Monterey's Calculator app with both **Basic** and **Scientific** modes.

![Calculator Screenshot](screenshot.png)

## Features

- **Basic mode** — standard arithmetic (+, −, ×, ÷), percent, negate
- **Scientific mode** — sin, cos, tan, log, ln, √, x², x³, 1/x, 10ˣ, π, e, n!, |x|, parentheses
- **Glassmorphism design** — blurred backdrop, dark theme, macOS-style window controls
- **Keyboard support** — type digits, operators, Enter for equals, Escape to clear
- **Responsive layout** — works on mobile and small screens
- **Ripple effects** — click feedback on all buttons

## Usage

Just open `calculator.html` in any modern browser — no server needed.

### Keyboard Shortcuts

| Key | Action |
|---|---|
| `0-9` | Digits |
| `.` | Decimal |
| `+` `-` `*` `/` | Operators |
| `Enter` or `=` | Equals |
| `Escape` | Clear |
| `Backspace` | Delete last digit |
| `%` | Percent |
| `(` `)` | Parentheses (sci mode) |
| `p` | π |
| `e` | Euler's number |
